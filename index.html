MircoOS

About

MicroOS this is an operating system (OS) for micro:bit of any version and color

Features

  • Simple and easy to use
  • Fast
  • Closed source (Don't import hex file in makecode...)
  • Free

Installation

1. Download the latest version of MircoOS

2. Connect your micro:bit to your computer

3. Drag and drop the downloaded file to the micro:bit

4. Wait until the file is copied

5. Disconnect the micro:bit from the computer

6. Turn on the micro:bit

7. Enjoy!
